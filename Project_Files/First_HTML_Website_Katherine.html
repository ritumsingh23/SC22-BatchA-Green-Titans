Support Vector Regression (SVR)

History:

This machine learning model was invented by Vladimir N. Vapnik and Alexey Ya. Chervonenkis in 1963

How it Works:

SVR is a supervised machine learning algorithm that can be used for classification and regression. It finds a hyperplane that divides a data set into different classes. The goal for this method would be to choose a plane with the greatest margin between the hyperplane and a data point.

Pros and Cons:

Pros Cons
-Thrives in High Dimension -Not good for big datasets because training will take a long time
-Support Vector Machines are all about choosing the right kernel with the right parameters and this can provide lots of flexibility -Less effective on messsy data sets with overlapping classe
-Fast Prediction -SVMs are not the most efficient algorithms and it can be quite costly computationally to train them
-SVMs can be used to both classify data and also predict continuous numerical values -Can be hard to interpret
-Have to scale data
