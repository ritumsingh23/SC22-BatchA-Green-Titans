Random Forest

What is Random Forest?

Random forest is a machine learning model. This model uses a bunch of decision trees to put in the input and calculates the output based on the mean or mode of the predictions made by the decision trees.

What is a Decision Tree?

A decision tree is another machine learning model which takes the input and predicts the output through(wip)

Pros and cons of Decision Tree

    Pros:
  • can be used for classification and regression
  • works well with non-linear data
  • works efficiently with large datasets
  • has a higher level of accuracy compared to decision tree
  • has many parameters
    Cons:
  • does not work well with linear data
  • is known to overfit
  • slow training

Types of Parameters

  • n_estimators
  • max_features
  • max_depth
